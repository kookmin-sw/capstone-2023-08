## ⚡ 프로젝트 소개

**"집 밖을 나가지 않아도 다양한 옷을 직접 입어볼 수 있습니다!"**

옷을 사러 나가지 않고 인터넷 쇼핑몰에서 사는 것이 평준화된 요즘, 직접 입어보지 않으니 옷 구매에 실패한 경험이 많이 있습니다.
저희 팀은 이 점에 주목하여 **바로 지금! 👀** 보고 있는 옷을 입어보고 만족스러운 구매를 위한 가상 피팅 서비스를 기획하였습니다.
